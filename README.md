# Aplicación Web – Agencia de Eventos

## Descripción
Este proyecto es un proyecto de una aplicación web desarrollado como parte del curso Taller de Programación.  
El proyecto se realizó en equipo de 5 personas, simulando un entorno profesional de desarrollo.  
Se diseñó y desarrolló una web app para la gestión de eventos de una agencia, incluyendo funcionalidades de creación, edición y visualización de eventos, así como manejo de usuarios y reservas.  

El proyecto se implementó utilizando Java, HTML, CSS, se gestionó con Maven y se desplegó en Apache Tomcat.

---

## Tecnologías
- **Lenguajes:** Java, HTML, CSS, JSP  
- **Herramientas:** Maven, Git, Apache Tomcat  
- **Conceptos:** Programación Orientada a Objetos (POO), MVC, desarrollo web, integración y despliegue
